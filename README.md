## Мой вклад в проект

Вся техническая и геймдизайнерская часть проекта лежала на мне.

В рамках разработки были реализованы:

- Покупка и расположение построек по сетке
- Поиск пути у врагов и дружественных солдат
- Игровой цикл с условиями победы и поражения
